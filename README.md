# Python Calculator

A command-line calculator built with Python.

## Features

### Basic Operations

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Modulus (%%)
* Floor Division (//)
* Exponentiation (**)

### Advanced Operations

* Square Root (√)
* Factorial (!)
* Absolute Value (abs)
* Percentage (%)
* Logarithm (log)

### Statistical Operations

* Average (avg)
* Maximum (max)
* Minimum (min)

## Error Handling

The calculator handles common errors such as:

* Division by zero
* Modulo by zero
* Floor division by zero
* Negative factorial
* Negative square root
* Invalid logarithm values
* Invalid user input
* Empty lists for Average, Maximum, and Minimum

## Requirements

* Python 3.x
* Built-in math module

## How to Run

Open a terminal and run:

```bash
python calculator.py
```

## Example

```text
Enter Operator: +

Enter First Number: 10
Enter Second Number: 20

Sum: 30
```
